**What is still true and unowned**

| Piece | Home |

| Ladder promises as flux | `#form-ordinum-governs-flux-web` |
| Manifest vs ordinum (no regula) | `#form-manifest-prescribes-vivium` |

**Residual claim (sole purpose of a thin FE):**

1. The **kernel** is the only intended *imperative* surface of a nomos.  
2. Identity, epistemic tags, deps/consumes/promises, conservation, assumptions, and (as they land) lifecycle / coupling class / phase binding are **declaration data**.  
3. Moving a concern out of the kernel body into the declaration is how new audits fall out for free (permit-void, multirate, maturity) — without inventing a kernel DSL (DECLARATIVE-FRONTIER failure record: rejected).

**Not claimed:** that every backlog field is built; that `NomosDecl` is complete; five-layer scaffold as law; agentic “assessed” rung as store truth.

**Authority honesty:** spike attributes the steer to Joseph (2026-07-12). No dedicated `DECISIONS[…]` row found this pass. Segment epistemic status should say **Joseph-steer (spike-attested) / project method**, not invent `:by us` unless parent or DECISIONS confirms.

#### Proposed FE (only if promote)

**Slug suggestion:** `#form-kernel-imperative-boundary`  
**Type:** formulation · **Aspect:** Project / Machine · **Depends:** `#def-nomos`, `#norm-declaration-must-convict`, `#form-nomotheke-registry`  
**OUTLINE:** §II near nomos / nomotheke / flux web.

1. **Imperative boundary.** For each nomos, the **kernel** (numerical procedure) is the sole intended imperative escape hatch. Everything else about the article — identity, epistemic self-description, interactions, lifecycle stage, and coupling class — is **declaration data** the audit may read without running the kernel.
2. **Direction of progress.** “More declarative is better” means **extracting** implicit contracts from kernel bodies into the nomotheke (and related artifacts), not rewriting arithmetic as udon. A DSL for the kernel’s numerics is **out of bounds** (re-invents a worse language).
3. **Audits fall out of data.** Graph queries over declarations (requisite chain, vocabulary closure, conservation flags, weakest-link fold, and — when declared — statistic match / timescale band / execution class) are the pre-run picture of honesty. Kernels run after the graph is sound or after honesty is reported.
4. **Compliance debt is named, not denied.** Fields still only in kernel bodies or design prose (e.g. execution-class, timescale-band, stage lifecycle, phase binding; source-derived versions) are **residual debt** under this law — absence from code does not revoke the boundary.

**Stage:** `draft`. **Status:** exact *as direction* if parent accepts Joseph-steer attribution; else robust-qualitative until DECISIONS row.

**Do not absorb:** full Layer 0–4 scaffold map; illustrative udon world-building; “kinds of nomoi” taxonomy essay; POSTURE / slot probation (already collapsed into manifest/ordinum).
