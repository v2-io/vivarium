# `.super-archive/` — fully integrated or superseded substrate

*Not claim canon. Not a tactical dependency. Not a second OUTLINE.*

## What this is

Material that has **already been promoted into claim homes** (`core/src/` segments, LEXICON, DECISIONS, ETHICS, ordinum, ASSUMPTIONS) and/or is **superseded by present segment truth** — so it no longer needs to sit in live `doc/design/`, live theory dual-homes, or the mineable residual of `.archive/`.

| Layer | Role |
|-------|------|
| **`core/`** | Sole claim truth |
| **Live `doc/`, `msc/` spikes, DECISIONS** | Active sources, instruments, decision ledger |
| **`.archive/`** | Sunset material that may still hold **unmined gold** or is held as re-founding archaeology pending residual extract |
| **`.super-archive/`** | **Done mining for claims** — integrated into segments, or honestly superseded; keep for provenance / rewind only |

## Graduation rules (all must hold)

1. **Claim homes exist** for every load-bearing assertion the file was carrying — or the file is pure process/history with no remaining claim residue.
2. **No tactical dependency** from main tree code or segments (same as `.archive/` accounting).
3. **MANIFEST entry** names: path moved, date, superseding slug(s) or "pure ice / re-founding specimen."
4. Prefer **`git mv`** so history survives; update any live pointers that still cited the old path.

## What does *not* go here yet

- Design/theory files still carrying unextracted gold (even if partially dual-homed).
- Spikes whose **measurements** are still the primary instrument (e.g. `msc/spike-wavelet-store/` until cited numbers live only in segments — RUN.txt stays with the spike until then).
- `.archive/memory-surfaced-*`, PHASES, VIVARIA-* until residual EXTRACT list is empty.
- Anything an agent might still need as **source material** for a named OUTLINE gap.

## Relation to audits

`audits/.integrated/` is for **audit reports** after verified/adjudicated/integrated (`#ops-audit-integration`). This directory is for **design/archive/prose substrate**, not audit packets.
