# Vivium

A **vivium** is one instantiated world-artifact that can be named, forked, and cited — not the engine that builds worlds.

## Formal Expression

A **vivium** (pl. *vivia*) is one concrete world: the build-stack and law that generate it, the target world state, its backstory, and its interventional history, individuated so that it can be stored, forked, and cited as a single object.

**Not** *vivarium* — that word names the project, the class of such engines, and this repository.
