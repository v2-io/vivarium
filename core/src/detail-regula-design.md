# Regula design — world-level conformance as data

The regula is the world-level conformance profile a vivium declares: slots filled at minimum rigor, permitted absences, epistemic posture. Ordinum is the phase floor; regula pins it. Held design (largely unbuilt as `regula.rs`); naming Joseph-settled 2026-07-11.

## Formal Expression

1. **Honesty stack grains.** ASSUMPTIONS (constants) · nomos declaration (algorithm) · **slot** (system-role by flux interface) · **regula** (world conformance). Same pattern: data → keys/tests → audit is computation.

2. **Ordinum.** Per-phase Charge/Promise floor as data (`tabularium/terrestris.ordinum.udon`). PHASES prose was the reportatio (now ice provenance only — live law is the ordinum). Pin `ordinum@version`; PHASES/ordinum revision = new version; regulae keep pins, opt in by re-versioning.

3. **Permits and requisite closure.** Permit = (absent slots, void condition on consumed quantities). When a consumer voids a permit, slot must fill before consumer lands. Requisite tree is a planning query (`vivarium audit --add` shape). Specimen: rain without atmosphere.

4. **In-vivia results as memos.** Result = query + analysis nomos with complete dependency cone; invalidation flags touched results; hash compare flags *changed* results. Posture caps claims; cone-fold wins. Sufficiency vs absence asymmetry (settled with Joseph 2026-07-11).

5. **Regula lineage.** Each `regula@version` immutable/citable; head = current best. Status reports conformance-to-pin and gap-to-head.

6. **Promise conformance.** In-force promises = union minus defeasances; maturity ladder named→specified→claimed→kept; no-predicate ⇒ cannot mark fulfilled; defeased-but-still-consumed = error.

7. **Two chapters.** (1) Fidelity — slots, minima, permits, posture. (2) Participation — access matrix, engagement modes, realizability gates, stewardship. **Moratorium supremacy:** not within regula's gift; audit checks program law ( #scope-moratorium-endogenous-emergence ).

8. **Naming (settled).** regula/regulae; Regula Terrestris; Ordinum (*Ordinatio Ordinum*); Greek law-article layer vs Latin governance layer vs English lived concepts.

9. **Critiques held.** Prose-wish (must compute); prematurity (live specimen exists); slot granularity = flux interface; floor drift resolved by pin; posture overreach (fold wins); first Terrestris audit forces biosphere stand-in or permit (Joseph).
