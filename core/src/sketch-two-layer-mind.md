# Two-layer mind (exploratory — not primary)

*Exploratory sketch: novel agent architecture idea that does **not** supersede existing segments and is **not** ready for primacy. Source peel from `doc/design/DESIGN.md`. Not Level-C agent-seam work; not a claim that this is vivarium law.*

## Formal Expression (sketch)

1. **Fast layer** — every tick, many agents: pure formal AAT/ASF dynamics (needs, adaptation, GUC). Numeric, deterministic, authoritative substrate state.
2. **Slow layer** — rare, few agents: LLM (or peer mind) only at decision-theoretic *aporia*, not on a clock. Output may **perturb** formal state (re-weight a goal, form an edge), never drive motor actions by bypassing the fast layer.
3. **Moral scope.** Fast layer alone is framed as out of moral scope (Class-1 separation-by-construction — ETHICS). The moment an LLM is in the loop, ETHICS and moratorium constraints bind.
4. **Open empirical.** Whether LLM perturbations remain identifiable for *in vivia* measurement is **unverified** — do not assume either way (#disc-aat-vivarium-object-map adjacency).

## Epistemic Status

**Max attainable:** empirical / discussion for the architecture; **currently `sketch`**. No DECISIONS row; design prose only. Level C gates any experiment design.
