# Adding a world-system — the six-clause contract

To add a world-system and have the machine compose it, declare its place in the algebra. Three walls keep authors from breaking each other: flux interface, complete key, and execution-class + multirate coupling.

## Formal Expression

1. **$R$ / $L$ / closure.** Macro summary it exposes, how it materializes detail, and **fated** closure so it is memoizable ( #form-rl-closure-algebra · #post-determinism-as-ontology ).

2. **Fluxed quantities.** Per-quantity coupler interface it produces/consumes — the *only* thing others may depend on; fine-grained, never a monolithic blob ( #form-flux-web ).

3. **Execution class and timescale band.** So coupling treats it quasi-static or time-averaged (batch-deep / relaxation / procedural-tight; #form-scale-separation-directional · #form-kernel-imperative-boundary · NomosDecl `ExecutionClass` / `Timescale`).

4. **Determinism in keyed inputs**, with declared nomos-version (hand-stamped today; source-derived is the target under #form-complete-content-addressed-key ).

5. **Nomotheke declaration** ( #form-nomotheke-registry ): epistemic tags, deps (weakest-link fold), promises with conservation + statistic/exactness, assumption anchors into `ASSUMPTIONS.md`, structure/geometry/unphysical-term boxes as fields allow. Declaration mints store keys — undeclared nomos cannot reach world-law.

6. **Regime probes written first** ( #norm-regime-probes ) — including seam probes (space continuity; time near-stationarity). A declared tier is a falsifiable claim; the probe convicts.

7. **CHONK prior-art validation** (Gailleton et al. 2024, primary-read): geoscience LEMs break when processes are combined by superposing independently-solved flux grids under interdependence. Decomposition worth adopting: numerically separate **`{fluxes, properties, processes, graph}`** as loosely-coupled modules — our fluxed quantities ≈ their fluxes; systems ≈ processes; tile/$R$/$L$ ≈ graph; column/strata ≈ properties. (a) Graph process-agnostic, processes domain-specific. (b) Every interdependent process at a node runs to **definitive** fluxes before transmitting downstream — sufficient-statistic seam at cell grain. Caveat: CHONK is single-resolution without memoization/LOD — informs coupling/seam only.

8. **Domain-fixation guard (Joseph, 2026-07-10).** Framework is a **domain-agnostic simulation substrate**. Field-on-a-grid is not the assumed primitive. Erosion/hydrology are early *instances*, not the subject. Validate generality across representation *kinds*, not only geomorphic systems.

## Epistemic Status

**Max attainable: exact** once fully mechanized on NomosDecl. **Currently `robust-qualitative`:** checklist is standing design contract; registry clauses partially built; execution-class/timescale fields exist; CHONK is primary-read support. Stage `draft`.

## Discussion

Without this contract, "add a system" means a function and a CLI flag. With it, interdependence is mediated, never shared-mutable.
