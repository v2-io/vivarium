# Agent-seam constraints (late-phase holder)

Higher-order constraints for the agent seam are held here so world-building does not foreclose them — without treating them as current geology work.

## Formal Expression

**Phase.** The agent seam (cognitive LOD, two-layer mind, agent observation / action channels, save / fork for inhabited worlds) is **Phase 7 / Aboriginal-class work** relative to the Terrestris ladder: primary work now is exogenous exploration and world law, not building minds into the vivium. **Nothing in this segment is primary work until Level C is cleared** ( #scope-asf-reading-gates). The standing moratorium still binds ( #scope-moratorium-endogenous-emergence · `ETHICS.md`).

### Constraints (held, not all built)

1. **Two-layer mind as wrapping.** When the seam is built, the slow LLM layer is the Coupled component and the fast formal layer is the wrapper that restores directed separation at agent level ( #lexicon/term/wrapping-regime · #asf/1-aat/der-class-coercion-via-wrapping · #asf/3-llm/der-logogenic-as-wrapping · #asf/1-aat/der-directed-separation). **Prefer W₁ (strict) wrapping** where the interface is fully under our control. The (C2′) condition — no goal-correlated cross-call state that breaks the structural bound (see also #asf/1-aat/disc-w1-structural-bound-boundary) — is what makes W₁ a design choice rather than a gift: aporia / cross-call state must be stateless or state-partitioned, or only a behavioral (W₂-type) bound remains. *Unbuilt; discussion-grade design constraint.*

2. **Assumptions satisfied by construction (thin).** The formal layer can be authored Separated; the action space can be the agent's entire external interface (bounded signaling exact if no leakage channels are added); GA-1-style housing honesty is a build obligation if observation channels are authored. *Conditional on building those channels honestly — not a present measurement.*

3. **Ground-truth AAT quantities measurable here (thin).** $U_M$, $\delta_t$, $\eta^\ast$, $\mathcal{T}$, $\rho$ per region, per-dimension mismatch — in principle exactly computable when we hold $\Omega$ and $\theta$ ( #disc-aat-vivarium-object-map). Wild estimation lives at #asf/3-llm/obs-evaluation-metrics; vivarium's role is measurement by construction. Behavioral $\hat\kappa$ probes inherit the confound named in #asf/1-aat/der-directed-separation (hold $M_{\tau^-}$ fixed across goal conditions). *Capability claim for the laboratory design; not "already instrumented."*

4. **Per-agent fated RNG before parallel agents.** World fated-noise discipline is realized for world law (`vivarium-world` noise keyed on $(\mathrm{seed}, k)$). Paths that still use a stepped stream keyed on step count break under concurrent agent steps. **Per-agent splittable / fated seeds are a prerequisite of the seam**, not a cleanup after parallel agents ship ( #post-determinism-as-ontology known incomplete surface: agent-layer RNG).

5. **Fork co-development with ASF.** Save / fork machinery for inhabited worlds should be developed against #asf/4-eli/hyp-checkpoint-forking-failure-modes (`discussion-grade` upstream: forking locally cheap, systemically catastrophic — identity bifurcation, accountability fragmentation) and #lexicon/term/forks — currently unreconciled across the two projects. *Pointer, not a vivarium fork theorem.*

6. **Crèche = locus only, for now.** ASF names AAT-grounded experiential training environments ( #asf/3-llm/hyp-experiential-training). Vivarium is deliberately only the **locus** where such an environment could exist — not a crèche implementation. `ETHICS.md` states the crèche tier's demands; the AAT shape of the environment is not specified here. **Decisions about the agent seam must not foreclose the possibility; nothing may assume the crèche is already the project.**
