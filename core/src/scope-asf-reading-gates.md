# ASF familiarity gates for vivarium work

Working on vivarium requires cumulative ASF familiarity: Level A every session, Level B before designing a world system, Level C as a hard gate before any agent-seam work.

## Formal Expression

Levels are **cumulative**. "Read" means read-for-comprehension. Work that lands in the ASF tree is bound by ASF's own SOP set (`asf/doc/sop/agents.sop.md` and its CLAUDE/AGENTS/GEMINI entry points), not by vivarium's conventions.

### Level A — every session, before any vivarium work

1. This project's front door: `CLAUDE.md`, `core/OUTLINE.md`, `FORMAT.md`, and `#scope-moratorium-endogenous-emergence` / `ETHICS.md` (Standing Moratorium Imperative).
2. Know that vivarium is an ASF instrument: worlds authored so AAT quantities can be known by construction ( #disc-vivarium-purpose, #def-in-vivia), with mathematics cited as `#asf/…` rather than re-proved here.
3. Know where the theory lives: `~/src/archema-io/asf/` (AAT / TST / LLM / ELI volumes). Cost: minutes.

### Level B — before designing any world *system*

A "world system" here means a phase's physics, climate, ecology, or the run-modes / provenance machinery that will host later agents. Read:

1. ASF root orientation (disposition + conventions) via `asf/CLAUDE.md` (or the SOP it routes to).
2. `01-aat-core/OUTLINE.md` Part I (the adaptive-systems spine).
3. The three Part-I anchor segments: #asf/1-aat/result-mismatch-decomposition · #asf/1-aat/result-persistence-condition · #asf/1-aat/def-adaptive-tempo.
4. Skim `asf/spikes/epistemic-target-ontology/00-spike.md` (typed ontology of law / state / chance / compute-shortfall that the vivarium object map leans on — #disc-aat-vivarium-object-map).

Cost: an attentive hour. The point is knowing what $\rho$, $\theta$, and the persistence condition demand of a world that will host agents — which changes how weather and other systems are built.

### Level C — before touching the agent seam

The agent seam includes cognitive LOD, the two-layer mind, agent observation / action channels, and save / fork semantics for inhabited worlds. **This level is a hard gate: the agent seam is load-bearing for the project's laboratory purpose, and it must not be built from an unread theory.** Add:

1. #asf/1-aat/der-directed-separation in full (GUC classes, W₁/W₂, bounded-signaling).
2. `03-llm-core/OUTLINE.md` (scope lattice + bias bound).
3. `04-eli-core/OUTLINE.md` preface + the deaths-and-defenses material (outline §04.3 region), together with `ETHICS.md`.
4. `asf/spikes/epistemic-target-ontology/01-ga1-verification.md` (housing choices in agent perception are made knowingly).

Cost: a day of real reading. Moratorium ( #scope-moratorium-endogenous-emergence) still binds: Level C does not authorize endogenous frontier / emergence-capable minds. After Level C is cleared, design constraints for the seam itself are held in #scope-agent-seam-constraints (forward holder, not a prerequisite of the gate).

### Recurring dives (per-cycle gate)

At natural junctures — opening a new world system, closing a spike, promoting a design claim — ask: *which ASF segment is this an instance of?* If the answer is "none", that is either a gap worth reporting upstream (**vivarium is allowed to find holes in AAT — that is half its value**) or a sign the work has drifted from the laboratory bet. Write the answer where the work lives.
